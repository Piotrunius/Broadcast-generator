üì° Broadcast Generator

Simple Mode

The simple broadcast generator allows you to quickly create standard broadcasts for common scenarios:

  1. Select the SCP breach event from the checkboxes (035, 008, 409, 701)
  2. Choose the threat level (HIGH, MEDIUM, LOW)
  3. Select the site status (SCP BREACH, SITE LOCKDOWN, etc.)
  4. Pick the message length (LONG, SHORT, MINIMAL)
  5. Click GENERATE to create your broadcast
  6. Copy the generated message to use in-game
üí° Pro Tip:

The character counter shows your remaining characters (200 max). Keep broadcasts concise for better in-game communication.

Advanced Mode

Advanced mode provides additional features for power users:

  • Support for more SCP types (076, 323, 610)
  • Custom message modification
  • Number-free formatting to avoid Roblox tagging
  • Requirement authorization options
  • Hidden Easter eggs (Konami code compatible!)

How to Use

  1. Enter an input item in the search field
  2. Optionally enter an output item (leave blank to see all possible outputs)
  3. Press CALCULATE RECIPES or hit Enter
  4. View the refinement path and settings needed

Keyboard Shortcuts

TAB - Autocomplete with first matching element ENTER - Calculate recipes ESC - Clear all fields
üí° Pro Tip:

The highlighted settings (ROUGH, COARSE, 1:1, FINE, VERY FINE) show which refinement type to use for each step.

Recipe Data Source

All recipes are sourced from the official SCP-914 Wiki and are regularly updated to match the game.

Performance Mode

Performance mode disables animations and transitions to improve performance on lower-end devices. All functionality remains intact, only visual effects are reduced.

  • Toggle in the Settings page
  • Persists across page visits
  • No impact on core functionality

Audio Controls

The application includes subtle sound effects for button interactions. You can toggle these on/off in Settings.

Dark Theme

The interface uses a dark cyberpunk theme inspired by the SCP Foundation aesthetic, optimized for long sessions and reduced eye strain.

üîß Troubleshooting

Broadcast Not Generating

Ensure you have:

  • Selected at least one SCP (if applicable)
  • Chosen a threat level
  • Selected a site status

Performance Issues

If the site feels slow:

  • Enable Performance Mode in Settings
  • Close unused browser tabs
  • Try a different browser (Chrome/Firefox recommended)
